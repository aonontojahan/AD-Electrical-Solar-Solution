# AD Electrical & Solar Solution

A modern, fully responsive website for AD Electrical & Solar Solution - a complete solar energy solutions provider.

## Overview

This website showcases solar installation services for residential, commercial, and industrial clients. It includes multiple pages with service details, project galleries, and contact information.

## Features

- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **Multi-page Structure**: Home, About, Services, Projects, and Contact pages
- **Project Gallery**: Showcase of completed solar installations
- **WhatsApp Integration**: Direct WhatsApp chat buttons for instant communication
- **Modern UI/UX**: Clean design with green/yellow color scheme
- **Fast Loading**: Optimized images and lightweight code

## Pages

1. **Home** - Landing page with hero section, services overview, and project previews
2. **About** - Company information, core values, and team members
3. **Services** - Detailed service offerings and process explanation
4. **Projects** - Gallery of completed solar installations
5. **Contact** - Contact information and communication channels

## Technology Stack

- HTML5
- CSS3 (Custom CSS with CSS Variables)
- JavaScript (Vanilla JS)
- Font Awesome Icons
- Google Fonts (System fonts)

## Installation

1. Clone or download the repository
2. Open `index.html` in your web browser
3. No build process required - it's a static website
